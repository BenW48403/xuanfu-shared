**二、UI分类说明**

> UI分为两类，审批机制不同。通用UI一旦审批锁定即为全系列标准；谜题UI逐套单独审批。

|            |                                                                                                                                                                                                                       |                    |                                          |
|------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------|------------------------------------------|
| **UI类型** | **定义与元素列表**                                                                                                                                                                                                    | **审批节点**       | **跨章节复用**                           |
| **通用UI** | 全局常驻元素： ・底部道具栏 ・左上角设置齿轮按钮 ・右上角提示「?」按钮 ・场景内「?」热点标记样式 ・提示菜单栏图标组（灯泡/影片/锁头/红叉） ・「◁」退出谜题模式按钮 ・NPC对话框 ・死亡/通关遮罩 ・《玄符秘录》入口图标 | **R-12a 一次审批** | ✓ 全系列标准，后续章节直接复用           |
| **谜题UI** | 每个谜题专属的交互界面： ・密码锁键盘 ・描画操作界面 ・方向选择罗盘 ・供奉顺序三炉 ・铜镜照字效果 ・提示内容展示界面（占位状态）                                                                                      | **R-12b 逐套审批** | 每套提交：起始状态图+最终状态图+文字说明 |

> 通用UI审批通过后，其视觉风格（包括「?」热点标记样式、提示菜单栏图标组）锁定为全系列标准，归档至D-10美术风格指南。
