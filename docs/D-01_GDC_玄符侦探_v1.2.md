**二、核心玩法**

**2.1 游戏类型**

点击式密室解谜冒险游戏（Point-and-Click Escape
Adventure）。玩家扮演大学生林砚，接受灵异委托进入密闭场景，通过收集道具、解开谜题、与NPC交互，找到逃脱方法，同时推进寻找失踪爷爷的主线。

**2.2 核心循环**

- 进入场景 → 探索收集道具 → 发现谜题线索

- 解开谜题 → 解锁新区域或获得新道具

- 满足NPC条件 → 通过门卫 → 推进主线

- 触发死亡陷阱 → 回到上一步 → 习得教训再试

- 完成全部谜题 → 逃脱成功 → 获得主线线索碎片 → 引出下一章

**2.3 核心卖点**

- 中式灵异美学：区别于西方恐怖游戏，以民间符咒、怨灵、风水为核心元素

- 叙事驱动：每章密室独立成章，共享主线，玩完一章想玩下一章

- 高度模块化：章节结构公式化，支持Agent团队快速批量开发

- 通勤友好：单章60-90分钟，可分段保存，适合碎片化游玩

**2.4 差异化亮点**

- 现有市场上密室解谜游戏普遍使用西方恐怖风格，中式灵异赛道竞争稀少

- 系列叙事结构让玩家有持续回访动机，而非一次性体验

- Agent团队开发模式支持低成本快速迭代，可在验证市场反应后快速扩充章节

**三、目标用户**

**3.1 核心用户画像**

- 年龄：18–35岁

- 设备：智能手机（Android / iOS）

- 场景：通勤、等待、睡前碎片化时间

- 偏好：解谜、悬疑、国风、剧情向游戏

- 平台习惯：抖音日活用户，有小程序使用习惯

**3.2 用户痛点**

- 现有解谜游戏剧情薄弱，缺乏持续吸引力

- 恐怖游戏多为西方风格，与中国玩家文化背景有距离

- 精品密室游戏体量大，不适合碎片时间

**四、美术风格**

**4.1 整体定位**

半写实风格（Semi-Realistic）。保留真实的质感和比例，但线条有所简化，适合手机小屏清晰辨认。色调偏暗、饱和度适中，营造压抑神秘的灵异氛围。

**4.2 视觉参考**

- 人物风格：半写实，参考《阴阳师》NPC立绘的细节程度，但不使用日系漫画比例

- 场景风格：2.5D横版，多层视差，参考《青鬼》《Forgotten
  Hill》系列的空间感

- UI风格：中式古典元素融合现代简约，参考古籍装帧、朱砂印章、线描纹样

- 色彩基调：墨绿、赭石、朱砂红、烛火黄、月白为主色系

**4.3 技术规格**

|            |                                                     |
|------------|-----------------------------------------------------|
| **规格项** | **要求**                                            |
| 屏幕方向   | 横屏（Landscape），16:9比例                         |
| 分辨率     | 1080×1920px 设计基准                                |
| 场景层次   | 纯2D固定视角：单张背景图+可交互热点（Demo阶段）     |
| 角色/NPC   | 2D精灵图，单角度（正面或3/4侧面），Demo阶段纯2D方案 |
| 包体限制   | 单章 ≤50MB（TikTok Minis硬性要求）                  |

**五、项目规模与技术选型**

**5.1 开发模式**

多Agent全自动开发流水线（10个核心Agent）。人工参与点：章节主题确认、每份文档节点审核、最终上线决策。

**5.2 技术选型**

|            |                                                     |
|------------|-----------------------------------------------------|
| **技术项** | **选型**                                            |
| 游戏引擎   | Cocos Creator（TikTok Minis官方支持，Demo阶段纯2D） |
| 运行模式   | TikTok Native Runtime（比HTML模式性能更好）         |
| 图像生成   | Midjourney / Stable Diffusion（M1 Agent输出提示词） |
| 音频生成   | Suno / Udio / ElevenLabs（M3 Agent输出提示词）      |
| 账号系统   | TikTok静默登录（绑定TikTok账号，无需额外注册）      |
| 存档系统   | 云端存档绑定TikTok账号                              |
| 变现模式   | 广告变现（激励视频）+ 月卡免广告                    |
