**四、最终密码完整推导链（方案C：铜牌符文数字化）**

**4.1 设计逻辑**

最终密码9401的推导需要整合三个跨场景的线索，这是全章难度最高的谜题（★★★★★），要求玩家回顾并整合从入场到最后收集到的信息。

|----------|----------------------|-------------------------------------------------------|------------------------------------------------------------|
| **步骤** | **线索来源**         | **线索内容**                                          | **推导**                                                   |
| **①**    | **入口大厅展板**     | 「光绪三十年入藏，1904年」                            | 数字1904首次出现，玩家在此阶段不需要使用它，但应当注意     |
| **②**    | **P-11保险箱密码**   | 保险箱密码正是1904                                    | 1904被强化记忆——玩家刚用它开了保险箱，不会遗忘             |
| **③**    | **T-13馆长日记末页** | 「铜牌符文对应方位：东1南9西0北4」                    | 1904的四个数字被重新赋予方位含义——玩家意识到这是同一组数字 |
| **④**    | **T-12铜牌**         | 铜牌正面四个符文，从左到右顺序为：南符→北符→西符→东符 | 对照日记方位表：南=9，北=4，西=0，东=1，读出9401           |

**4.2 铜牌视觉设计要求（给M1）**

- 铜牌正面需要设计四个可辨认的方位符文（东南西北各一个）

- 符文风格：仿汉代篆书，简化为可识别的几何形态，不使用现代汉字

- 四个符文从左到右排列顺序固定：南符在最左，依次为北符、西符、东符

- 符文尺寸：在128×128px的道具图标上清晰可见，在512×512高清版上可辨认细节

- 铜牌背面：爷爷红笔批注位置在符文正上方，字体用现代手写风格形成视觉对比

**4.3 设计自查：三条P3原则核验**

|                    |                            |                                                                                                      |
|--------------------|----------------------------|------------------------------------------------------------------------------------------------------|
| **原则**           | **要求**                   | **本谜题执行情况**                                                                                   |
| **线索可见性原则** | 线索在同场景或相邻场景可见 | 日记（馆长办公室）和铜牌（地下室）虽不在同场景，但日记明确指向铜牌，构成「显式跨场景依赖」，符合规则 |
