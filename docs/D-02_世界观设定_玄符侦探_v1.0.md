**D-02 世界观设定文档（World View Bible）**

本文档是所有章节叙事的底层依据，跨章节一致性的唯一标准

文档版本：v1.0 \| 负责Agent：P2 叙事设计 \| 输出给：P2自用、M1、M3

**一、核心人物设定**

**1.1 主角：林砚**

|          |                                                                                                          |
|----------|----------------------------------------------------------------------------------------------------------|
| **属性** | **内容**                                                                                                 |
| 姓名     | 林砚                                                                                                     |
| 年龄     | 21岁，大三在读，专业：汉语言文学                                                                         |
| 外貌     | 身形偏瘦，五官清秀，常穿宽松棉麻衬衫，右手腕戴一串爷爷留下的旧念珠                                       |
| 性格     | 外表平静，内心敏感；遇事爱先观察再行动；表面上装作不信鬼神，骨子里对未知充满好奇；对爷爷有深厚的情感依赖 |
| 能力     | 文字理解力强，善于从细节中发现规律；初期对《玄符秘录》一无所知，随游戏推进逐渐习得基础符法               |
| 口头禅   | 「爷爷肯定知道这是什么意思……」；「先看看，不懂再说」                                                     |
| 行为禁区 | 不会主动攻击灵异存在，只求解谜逃脱；不轻信任何人（包括NPC）的第一句话                                    |

**1.2 爷爷：林鹤年（失踪人物，始终不出场）**

|          |                                                                                    |
|----------|------------------------------------------------------------------------------------|
| **属性** | **内容**                                                                           |
| 姓名     | 林鹤年，人称「林老」                                                               |
| 身份     | 私家侦探，暗地里是民间驱邪师，处理灵异委托已逾三十年                               |
| 失踪时间 | 游戏开始前两年，游戏内从未露面，仅通过遗留物、批注、旁人回忆呈现                   |
| 遗留物   | 《玄符秘录》（古籍）、事务所、旧念珠（林砚佩戴）、大量案件手记                     |
| 性格印象 | 稳重、寡言、处事周全；对林砚保护欲强，有意隐瞒职业真相；笔记里常有自言自语式的感慨 |
| 叙事作用 | 每章必须留下一条与爷爷相关的线索，串联主线；爷爷是「谜题答案」而非「故事角色」     |

> 设计原则：爷爷永远不能直接出场，他只能以「痕迹」的形式存在——红笔批注、泛黄照片、NPC对他的转述。这是本系列最核心的叙事张力来源。

**二、核心道具：《玄符秘录》**

**2.1 基本设定**

《玄符秘录》是一本外表普通的线装古籍，封面无字，内页混杂文言符法记录与爷爷的现代批注。它是林砚与爷爷之间最重要的纽带，也是每章解谜的隐性线索来源。

**2.2 功能规则（设计约束）**

|                          |                                                                              |
|--------------------------|------------------------------------------------------------------------------|
| **规则**                 | **说明**                                                                     |
| **每章必须出现一次**     | 林砚在某个关键时刻翻开秘录，对照场景中的线索，触发主线钩子                   |
| **不能直接给出谜题答案** | 秘录只提供「方向性提示」或「世界观解释」，不能成为解谜的直接道具             |
| **爷爷批注是关键内容**   | 每章的主线线索必须通过爷爷的红笔批注呈现，批注语言简短、隐晦、带有情绪       |
| **章节间内容递进**       | 每章翻到不同页数，随主线推进逐渐解锁更深的内容，第一章只能看前十页           |
| **视觉辨识度**           | 封面是深褐色旧布面，左下角有一个朱砂手印（爷爷留下的），这是本系列的视觉标志 |

**三、灵异体系规则**

本系列的灵异现象基于中国民间信仰，不引入西方魔法、恶魔崇拜等元素。所有灵异存在必须有明确的民俗依据或改编自真实民间传说。

**3.1 灵异存在分类**

|          |                    |                                                                            |
|----------|--------------------|----------------------------------------------------------------------------|
| **类别** | **典型形态**       | **行为逻辑与限制**                                                         |
| **怨灵** | 执念未消的亡者     | 只在特定范围内活动，有明确的「心愿」，满足心愿后自然消散；不能无故攻击玩家 |
| **物灵** | 年久有灵气的器物   | 被错误对待时才会反击，正确供奉或使用则平和；是谜题设计的常用载体           |
| **镇物** | 封印某种存在的器物 | 被破坏时释放被封印之物；爷爷留下的镇物是主线核心道具                       |
| **阴差** | 执行冥界职责的使者 | 不主动干涉阳间事务，但若阳间存在「失序」会短暂显形警示                     |

**3.2 符法基本规则**

- 符法由《玄符秘录》记载，普通人无法直接使用，需要林砚逐章学习

- 每章林砚只能使用一种符法，且该符法必须在本章内有完整的学习-使用过程

- 符法不是万能解法，只能辅助解谜，不能直接消灭灵异存在

- 符法的视觉呈现：朱砂色线条，书写时伴随轻微发光，不过度渲染特效

> 禁止事项：不得引入「作法驱鬼」的对抗性战斗；不得使用佛教法器作为主要元素（民间符咒与宗教法器有本质区别，避免引发敏感性争议）；不得出现直接的「地狱」「阎王」等词汇，改用「冥界」「幽途」等民俗表达。

**四、世界观地理：「遗忘之地」**

**4.1 基本概念**

「遗忘之地」是贯穿全系列的核心谜题——一个游走于阴阳之间的特殊空间，爷爷失踪与此有关。每章在某处会出现「遗忘之地」的线索碎片，玩家随主线推进逐渐拼凑出全貌。

**4.2 已知信息（第一章可知）**

- 是一个真实存在的地理位置，同时也是一种「状态」——被遗忘的人和事物会在此积聚

- 爷爷铜牌批注中首次提及，定性为「危险」

- 与某个特定的民间灵异事件有关，该事件爷爷曾经处理过

**4.3 禁止在第一章透露的信息**

> 以下内容属于后续章节的核心谜底，第一章中严禁任何Agent暗示或提前揭露：

- 「遗忘之地」的具体地理位置

- 爷爷是否仍然在世

- 爷爷失踪的真实原因

- 《玄符秘录》的完整来历

**五、叙事语言风格规范**

**5.1 整体基调**

压抑而不绝望，神秘而不猎奇。本系列的恐怖感来自「不该在这里的东西出现了」，而非血腥暴力或跳吓。叙事语言克制，留白多于直述。

**5.2 文案写作规范**

|              |                                                                  |                                                                |
|--------------|------------------------------------------------------------------|----------------------------------------------------------------|
| **场景**     | **推荐风格**                                                     | **禁止风格**                                                   |
| 环境描述     | 「烛台上的蜡烛已燃了大半，滴落的蜡油结成奇异的形状」             | 「这里好可怕，四周黑漆漆的让人毛骨悚然」                       |
| NPC对话      | 「你不应该来这里的……但你既然来了，就帮我做一件事」               | 「哇哦主角你来了！我需要你帮我！」                             |
| 死亡提示     | 「错误的选择。重来一次。」（简短、冷静）                         | 「Game Over！你死掉了！请重试！」（活泼跳脱）                  |
| 爷爷批注     | 「此物危。壬午年冬。——林鹤年」（简短、有日期、有情绪克制的判断） | 「这件东西非常危险，任何人都不能碰！」（过于直白）             |
| 主角内心独白 | 「爷爷见过这个……所以他知道。」（简短，带疑问，不过度解释）       | 「我突然想到了什么，原来爷爷早就预料到了这一切！」（过度感叹） |

**5.3 古文风格使用规范**

- 爷爷批注、《玄符秘录》正文可使用文言文，但必须通俗可解，不得使用生僻字

- 场景内的环境文案（标牌、便利贴等）使用现代白话文

- NPC对话根据角色身份决定：现代NPC用现代口语；年代久远的灵异NPC可用半文半白

- 文言文参考风格：《聊斋志异》白话版，而非先秦诸子散文

**六、主线时间轴（已知部分）**

|                    |                |                                                                        |
|--------------------|----------------|------------------------------------------------------------------------|
| **时间节点**       | **事件**       | **说明**                                                               |
| 壬午年（约30年前） | 铜牌案         | 爷爷处理过一个涉及铜牌的灵异案，留下批注「此物危」，细节不明           |
| 两年前             | 爷爷失踪       | 爷爷突然失联，失踪前一天给林砚打了一通电话，提到「书架最上层的书」     |
| 失踪前一天         | 最后的电话     | 「事务所的东西，以后就交给你了，守好它，别乱碰书架最上层的书。」       |
| 游戏开始时         | 林砚接手事务所 | 林砚半信半疑地经营事务所，偶然接到第一章的委托                         |
| 第一章             | 博物馆铜牌案   | 林砚在博物馆发现与《玄符秘录》记载吻合的铜牌，获得第一条爷爷失踪的线索 |
| 后续章节           | 待定           | 每章揭露一条新线索，最终拼凑出爷爷失踪的完整真相                       |

**七、跨章节一致性禁区**

以下设定一经确定不得在任何章节中修改或自相矛盾。所有Agent生成内容前必须核对本章节。

|                                |                                                        |
|--------------------------------|--------------------------------------------------------|
| **禁止事项**                   | **原因**                                               |
| **爷爷不得出场**               | 爷爷的「缺席」是叙事核心张力，一旦出场主线崩塌         |
| **林砚不得习得超自然战斗能力** | 他是侦探/解谜者，不是驱魔人，定位改变会破坏游戏类型    |
| **《玄符秘录》不得丢失或损毁** | 它是系列核心道具，必须全程陪伴林砚                     |
| **不得引入西方灵异元素**       | 吸血鬼、狼人、西方恶魔等与世界观冲突，破坏中式氛围     |
| **不得直接揭示爷爷生死**       | 这是系列终章的核心谜底，过早揭示则后续章节失去驱动力   |
| **不得让玩家直接消灭灵异存在** | 「消灭」改变游戏基调，本系列的解法是「理解」和「安抚」 |
