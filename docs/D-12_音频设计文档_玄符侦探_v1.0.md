**一、音频设计原则**

> 音频生产可与图像素材并行进行。提示词归档机制：审批通过的音频，其生成提示词归档至本文档及D-09.5，作为后续章节同类音频的风格基准。

- 氛围优先：BGM营造持续压抑的背景张力，不宜使用旋律性强的音乐

- 克制出现：音效在关键交互时出现，不滥用，保留沉默的空间

- 中国传统乐器：古琴/箫/编钟为主，禁止西洋管弦乐

- Stinger独立于BGM：主线钩子等重要时刻的Stinger音效必须在BGM之上清晰可辨

- 技术规格：BGM为MP3 128kbps循环，SFX为WAV 44100Hz 16bit，Stinger为MP3
  192kbps

**二、BGM设计**

|                           |              |                                        |                                                                                                                               |
|---------------------------|--------------|----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| **文件名**                | **时长**     | **适用场景**                           | **音频描述**                                                                                                                  |
| **bgm_ch01_default.mp3**  | 90-120秒循环 | 保安室/走廊/展厅A/修复室/祭祀室/办公室 | 古琴单音低频拨弦，间隔2-3秒一声，配合极低频弦乐垫底。无旋律，纯氛围，代表「正常但不安」的博物馆状态                           |
| **bgm_ch01_hall.mp3**     | 60-90秒循环  | 入口大厅                               | 比default更空旷，加入远处隐约的风声和门缝漏风声，应急灯电流嗡嗡声（低频）。强调「被困」的孤立感                               |
| **bgm_ch01_basement.mp3** | 60-90秒循环  | 地下储藏室                             | 几乎全静默，只有低频地底震动感（20Hz以下）、滴水声（随机间隔4-8秒一滴）、极远处隐约的古编钟单音。玩家需主动倾听才能察觉有声音 |
